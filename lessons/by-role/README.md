# 按角色的教训库

> 90+ 条实战教训,按 6 角色分类。每条 = 一条可执行规则。

## 索引

| 文件 | 教训数 | 来源 |
|---|---|---|
| `pm.md` | 15 | PRD 迭代 / 三端 review / 决策 |
| `frontend.md` | 20 | 某 B 端中台 V1 项目 + 某营销后台 联调 / 编码踩坑 |
| `backend.md` | 15 | 接口设计 / 数据库 / 联调配合 |
| `qa.md` | 12 | 4 层测试 / Bug 管理 / 自动化 |
| `design.md` | 16 | v4.0 规范 + 实战补充 |
| `agent.md` | 12 | One Click 方法论 + 自身设计 |

**合计**:90 条可执行规则

## 使用方式

- skill 的"常见踩坑"小节会按角色读取对应 md
- 手动查阅:`grep "关键词" lessons/by-role/*.md`

## 格式

```markdown
### #N {一句话规则}
**触发**:{何时应用}
**为什么**:{不这么做的后果}
**规则**:{具体做法}
```
