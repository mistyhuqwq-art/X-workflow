§ 03
EDITORIAL

Why margins matter

本期社论:为什么在设计里,留白不是"空",而是"意图"。

A grid is not a cage. 它不是把设计关起来,而是把决定权从直觉移到系统。做完这本期刊的 184 页,我们数过:每一页都遵守同一套 12 栏 + 16px gutter + 1.25 modular scale 的规则。

但读者读完后不会说"这期刊有规则"。他们会说"这期刊看起来安静,清晰,能让人读下去"。这是 Müller-Brockmann 1961 年的赌注,也是我们今天的赌注 —— 秩序的副产品是清晰,清晰的副产品是可信。

"The grid system is an aid, not a guarantee. It permits a number of possible uses, and each designer can look for a solution appropriate to his personal style. But one must learn how to use the grid; it is an art that requires practice."
— Josef Müller-Brockmann, Grid Systems, 1961
